This project performs **exploratory and time-based sales analytics** on a *Superstore* dataset using **Databricks**, **Apache Spark**, and **PySpark SQL**.

The solution is designed for:
- Business performance tracking
- Customer and order analysis
- Profitability and sales breakdowns
